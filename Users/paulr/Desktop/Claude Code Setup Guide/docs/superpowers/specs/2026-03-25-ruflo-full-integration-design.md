## 1. Executive Summary

**Notes on hook design decisions:**
- **Permissions**: Granular MCP allow list preserved (not collapsed to `mcp__*`). Explicitly intentional — `bypassPermissions` mode still benefits from an audit trail.
- **token-tracker.js**: Runs on ALL four tool types: Write/Edit/MultiEdit, Bash, Agent/Task. Full coverage restored.
- **SessionStart**: Uses `ruflo` (global install) instead of `npx ruflo@latest` to avoid cold-start timeout. Global install verified in Phase 1.
- **daemon start**: Uses `ruflo daemon start --headless` (global install, no npx overhead).
- **hook-handler.cjs session-restore**: Uses the local helper script (no npx cold-start).
- **PreCompact**: `context-persistence-hook.mjs` archives current state then returns exit code 2 to block auto-compaction. This exit-code-2 behavior is consistent with Claude Code's PreToolUse abort contract; Ruflo's ADR-051 applies the same convention to PreCompact. If Claude Code does not honor exit code 2 on PreCompact (edge case), archiving still succeeds — only the compaction-block is bypassed, not the data preservation.

**Preserved from current settings.json:** `env` block (all vars), `enabledPlugins` block, `effortLevel`, `autoUpdatesChannel`.

## 6. Context Autopilot — Infinite Context (ADR-051)

`context-persistence-hook.mjs` implements 3-hook infinite context pipeline:

```
Every UserPromptSubmit:
  → Archive turns to SQLite (SHA-256 dedup, importance ranking)
  → Track token count (actual API usage)
  → Update autopilot-state.json

Context reaches 85%:
  → PreCompact hook fires
  → Exit code 2 → BLOCKS auto-compaction
  → Archives current state first
  → Forces manual /compact only (user-controlled)

After /compact or SessionStart:
  → Restore top-ranked patterns from archive (4000 char budget)
  → Inject into additionalContext
  → Seamless continuation
```

**Thresholds:**
- Green: `<70%` — normal
- Yellow: `70-85%` — approaching limit, archive aggressively
- Red: `85%+` — prune stale entries, keep responses concise

**Storage:** `.claude-flow/data/transcript-archive.db` (SQLite WAL mode, ACID)
