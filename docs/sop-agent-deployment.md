# SOP - AI Agent Deployment

## Pre-Deployment
- Client assets collected
- CRM integration tested
- Booking system configured
- Knowledge base uploaded
- API keys secured in env vars

## Steps
1. Environment setup (clone, vars)
2. Build agent core (prompts, LLM)
3. Connect channels (SMS, chat, social, voice)
4. Configure integrations (GHL, booking, CRM)
5. Test and validate (flows, handoff, edge cases)
6. Launch and monitor 24h

## Post-Launch
- Monitor n8n/Make daily for 1 week
- Check dashboards for volume and quality
- Adjust prompts based on real conversation data
