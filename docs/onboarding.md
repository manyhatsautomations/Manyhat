# Client Onboarding

## Steps
1. Discovery Call (30-45 min) - understand lead flow, pain points, tech stack
2. Proposal - use calculator, send custom proposal with SOW
3. Kickoff - collect payment, schedule strategy call, gather assets
4. Build & Deploy - configure workspace, build flows, test thoroughly
5. Launch - go live, monitor 72 hours

## Timeline
| Phase | Duration |
|-------|----------|
| Discovery to Proposal | 1-3 days |
| Kickoff | 1-2 days |
| Build & Test | 3-5 days |
| Launch & Monitor | 3-7 days |
| Total | 1-2 weeks |
