# Client Onboarding Process

## Steps
1. Discovery Call (30-45 min)
2. Proposal - use ROI calculator, send SOW
3. Kickoff - payment, strategy call, collect assets
4. Build and Deploy - configure, test thoroughly
5. Launch - go live, monitor 72 hours

## Timeline
- Discovery to Proposal: 1-3 days
- Kickoff: 1-2 days
- Build and Test: 3-5 days
- Launch and Monitor: 3-7 days
- Total: 1-2 weeks
