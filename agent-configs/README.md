# AI Agent Configurations

System prompts, knowledge bases, and configuration templates.

- knowledge-bases/ - FAQs, objection handling, responses
- conversation-flows/ - Decision trees, flowcharts
- brand-voices/ - Tone and voice templates
- custom-clients/ - Client-specific overrides
